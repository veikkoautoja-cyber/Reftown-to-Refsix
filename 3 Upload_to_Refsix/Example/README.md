# Refsix Upload Examples

This folder will contain your converted Refsix upload files.

## Output Format

Converted files are named: `Export_assignments_toRefsix_DDMMMYYYY.xlsx`

Example: `Export_assignments_toRefsix_02Mar2025.xlsx`

## File Contents

The output file contains all required Refsix columns:

### Match Information
- Match ID, Competition, Venue, Date, Time, Time Zone

### Team Information
- Home/Away Team Name, Short Name, Colour (hex format)

### Game Parameters
- Team Size, Bench Size, No of Periods, Period Length, Interval Length
- Extra Time settings, Penalty Kicks settings

### Officials
- Referee, Assistant Referee, Assistant Referee 2, 4th Official
- Observer, Fees, Expenses

### Tags and Notes
- Tag 1-5, Notes (age group notes, game length notes, etc.)

## Usage

Upload the generated Excel file to Refsix using their import function.
